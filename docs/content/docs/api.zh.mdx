## 概述

此模板默认不提供公共 API。请将此页面作为记录您添加的任何端点的起点。

## 基础 URL

部署服务后设置此项，例如：

```
https://api.example.com
```

## 身份验证

在此描述所需的请求头或令牌（例如，Bearer 令牌或 API 密钥）。

## 错误格式

记录 API 返回的错误信封。示例结构：

```
{
  "error": "string",
  "message": "string"
}
```

## 端点

在实现时在此添加您的端点。
