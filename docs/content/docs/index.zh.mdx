## 快速入门

import { Step, Steps } from 'fumadocs-ui/components/steps';

<Steps>
<Step>

### 设置环境

使用 `uv` 创建并激活虚拟环境：

```bash
make setup
```

这将创建 `.venv` 目录并同步 `pyproject.toml` 中的所有依赖。

</Step>

<Step>

### 配置设置

复制示例环境文件并添加 API 密钥：

```bash
cp .env.example .env
```

编辑 `common/global_config.yaml` 自定义项目配置。

</Step>

<Step>

### 运行应用

执行主应用程序：

```bash
make all
```

或直接运行特定 Python 文件：

```bash
uv run python src/main.py
```

</Step>

<Step>

### 运行测试

通过运行测试套件验证一切正常：

```bash
make test
```

运行代码质量检查：

```bash
make ci
```

</Step>
</Steps>
